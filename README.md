Despliegue en Vercel
-------------------

Este proyecto se puede desplegar como un sitio estático en Vercel.

Pasos rápidos:

1. Instala la CLI de Vercel: `npm i -g vercel`
2. Desde la raíz del proyecto ejecuta:

```bash
vercel
```

o conecta este repositorio a Vercel en la web (se detectará la carpeta `public` automáticamente).

La configuración está en `vercel.json` y sirve los archivos en `public/`.
